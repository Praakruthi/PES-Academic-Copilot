# PES Academic Copilot

An AI-powered academic assistant for PES students that enables intelligent resource discovery, document-based question answering, summarization, quiz generation, and study assistance.

## Problem Statement

Students often spend significant time searching through notes, slides, question banks, assignments, and other academic resources on PES Academy. Finding relevant material and obtaining quick explanations can be inefficient.

## Vision

Build an AI-powered copilot that allows students to interact with academic resources using natural language.

Examples:

* "Find all resources related to Deadlocks"
* "Explain Round Robin Scheduling"
* "Generate MCQs from Unit 3"
* "Summarize this document"

## Planned Features

* Document-based Question Answering
* Intelligent Resource Discovery
* PDF/PPTX/DOCX Support
* Summarization
* MCQ Generation
* Quiz Mode
* Browser Extension Integration
